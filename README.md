# Node Modules In Memory

Put your `node_modules` into your memory for Linux (maybe MacOS) user has enough memory.

## Usage

``` bash
nmim start # start nmim from node_modules.tar
npim start --memory 500M

nmim save
nmim stop
nmim stop-all
```
